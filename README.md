# Library Management System

## Description

A Library Management System developed in C using a Singly Linked List and File Handling.

## Features

* Add Book
* Display Books
* Search Book
* Issue Book
* Return Book
* Delete Book
* Sort Books by ID
* Save Data to File
* Load Data from File

## Data Structure Used

* Singly Linked List

## Concepts Used

* Dynamic Memory Allocation
* Linked Lists
* Linear Search
* Sorting
* File Handling

## Language

* C Programming
